Live Examples

The following code snippets are live examples which you can edit and see the result immediately

1. Basic Ad Rotation (default config)

To initialize adRotator, you only need 2 parameters -
  • The Element where the ad should be displayed
  • The Ads.
The ads are provided in an array with the following format
[{url: '...', img: '...'}]

AdRotator also accepts a 3rd, yet completely optional parameter config which you can use to customize your Ads and its behaviour.

Then you can initialize and start adRotation as shown below

// initialise Ad Rotation 
var rotator = new adRotator(
  document.querySelector('.className'), 
  [{ url: 'xyz.com', img: '../xyz.jpg'}]  
);
// start rotation (returns undefined by default)
rotator.start();
Here is a live example demonstrating basic Ad Rotation with default Config. (To see the default config, you can uncomment the last line)
var rotatorDefault = new adRotator( document.querySelector(".defaultAdContainer"), [ { url: "https://niketpathak.com#0", img: "https://i.imgur.com/GsNCBNJ_d.webp?maxwidth=728&fidelity=grand" }, { url: "https://niketpathak.com#1", img: "./assets/img/square/square-1.jpeg" }, { url: "https://niketpathak.com#2", img: "./assets/img/square/square-2.jpeg" }, { url: "https://niketpathak.com#3", img: "./assets/img/square/square-3.jpeg" }, { url: "https://niketpathak.com#4", img: "./assets/img/square/square-4.jpg" }, ] ); rotatorDefault.start(); //rotatorDefault.conf;

2. Ad Rotation with custom config: Shape Leaderboard / Desktop-only

We pass a 3rd parameter config with the value { shape: "leaderboard"} to set the shape of the Ad as leaderboard. Doing this sets the height: 90px and width: 728px. Since this ad would be cut-off on a mobile device given its long width, we will also set the target: "desktop" so that this Ad would be visible on a desktop but hidden on a mobile device. 

// initialise Ad Rotation 
var rotator = new adRotator(
  document.querySelector('.className'), 
  [{ url: 'xyz.com', img: '../xyz.jpg'}],
  {
    shape: "leaderboard",
    target: "desktop"
  }  
);
// start rotation (returns undefined by default)
rotator.start();
A live example to go along with it. (Note: If you are simulating a mobile device, ensure you refresh the page)
